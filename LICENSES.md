## Paper, benchmark, and project notes

The Persome paper, benchmark suite, and publication project page are separate
works maintained outside this Runtime repository. They are not distributed by
this repository and are not licensed by its Apache-2.0 file. Each publication
repository must state its own code, data, model, and manuscript licenses before
release. A citation request in [CITATION.cff](CITATION.cff) is attribution
metadata, not a license grant.

## Personal data

Captured screens, derived memory, model exports, and user-provided fixtures are
the user's data. The project does not claim a license to personal data produced
by a local installation. Never submit real personal data in an issue, pull
request, benchmark, or test fixture.
